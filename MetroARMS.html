Metro ARMS Account Creation Guide

1. Request User Information
  • Send the Metro ARMS User Form (PDF) to the requesting user.
  • Instruct them to complete:
    • Page 2
    • Pages 6–8
  • Have the completed form returned to HPD-IT.
2. Submit Account Request
  • Rename the PDF to include the user’s last name.
  • Email Dawson Weijers (Dawson.Weijers@nashville.gov).
Text 
Subject: Metro ARMS Account Request – [Employee Name]

Body:
Please create a Metro ARMS account for [Employee Name].

The completed user form is attached.
3. Receive Account Credentials
  • Wait for the response email from Dawson.
  • Retrieve:
    • Username
    • Password
  • Provide credentials securely to the user.
4. Install Metro ARMS Software
Text 
Location:
Tech\General\Team Drive\MetroRMS

Folder:
2025 Install
  • Copy the zipped folder to the local machine.
  • Extract the contents.
  • Follow:
    • Oracle 11G Documentation
    • RMS v12.8 Installation Guide
5. Final Configuration
Text 
Required Info:
- Computer FQDN
- IP Address
  • Send the system details to Dawson.
  • This allows Metro ARMS access to be enabled.
